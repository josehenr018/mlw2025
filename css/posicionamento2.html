fixed

define o posicionamento de um elemento em relação à janela do navegador. Este elemento fica fixo na tela, mesmo que a página seja rolada, tornando-o ideal para elementos como cabeçalhos, rodapés ou menus que devem estar sempre visíveis.

static

Ao utilizar o position: static, o elemento segue o fluxo normal do documento, sem ser afetado por propriedades como top, bottom, left ou right

relative

A classe "relative" aplica a propriedade CSS position: relative ao elemento. Isso faz com que o elemento seja posicionado com base na sua posição original no fluxo do documento, mas permite que você mova o elemento usando as propriedades top, bottom, left e right. Ao aplicar position: relative, você pode ajustar a posição do elemento em relação à sua posição original.

absolute

É uma classe CSS que define o comportamento de posicionamento da caixa. Com "position: absolute", a caixa será posicionada de forma absoluta, com relação ao seu elemento pai que tenha "position: relative" ou "position: absolute". Caso o elemento pai não tenha nenhum desses posicionamentos, será considerado o corpo () do documento. A caixa também será removida do fluxo normal do documento, o que significa que outros elementos não serão afetados pela posição da caixa.

sticky

propriedade CSS position: sticky. Isso significa que a div "caixa" se comportará como uma div normal até que se atinja um ponto de rolagem definido. A partir desse ponto, ela se fixará na tela, como se estivesse em posição fixa, mas em relação à rolagem da página, e não em relação à tela como no position: fixed.
